MediCare - Healthcare Appointment System
A modern, responsive web application built with React and Material UI that allows users to find medical centers, browse specialties, and book appointments online.
Features
Search & Filter: Find medical centers by selecting specific States and Cities.

Appointment Booking: Select specific dates and time slots (Morning, Afternoon, and Evening).

Booking Management: Dedicated section to view and track all saved appointments.

Service Browsing: Quick access to various categories like Dentistry, Cardiology, Lab Tests, and more.

Responsive UI: Fully optimized for mobile, tablet, and desktop views.
